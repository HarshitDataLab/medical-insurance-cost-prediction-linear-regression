# 🏥 Medical Insurance Cost Prediction using Linear Regression

A Machine Learning web application that predicts individual medical insurance charges based on personal and health-related information using the Linear Regression algorithm. The application is built with Python, Scikit-learn, and Streamlit.

## 📌 Features

- Predicts medical insurance costs
- Built using Linear Regression
- Interactive Streamlit web application
- Fast and accurate real-time predictions

## 📊 Input Features

The model predicts insurance charges based on:

- Age
- Sex
- BMI
- Number of Children
- Smoker Status
- Region

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/HarshitDataLab/medical-insurance-cost-prediction-linear-regression.git
```

Move into the project folder:

```bash
cd medical-insurance-cost-prediction-linear-regression
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
